API Documentation

RESTful API endpoints for AutoML operations

POST /api/v1/upload

Upload dataset and get preview

Accepts CSV and Excel files
GET /api/v1/dataset/{session_id}/preview

Get dataset preview and statistics

Returns sample data and column info
POST /api/v1/dataset/{session_id}/detect-task

Detect task type (classification/regression)

Automatic task detection with confidence
POST /api/v1/train/{session_id}

Start model training process

Trains multiple models with hyperparameter tuning
GET /api/v1/training/{job_id}/status

Get training job status and progress

Real-time training progress tracking
GET /api/v1/evaluation/{session_id}

Get model evaluation results

Comprehensive performance metrics
GET /api/v1/evaluation/{session_id}/visualizations

Generate visualization URLs

Charts, plots, and model insights
POST /api/v1/reports/{session_id}/generate

Generate comprehensive report

PDF, HTML, or JSON format
Example Request
curl -X POST "http://localhost:8000/api/v1/upload" \
  -H "Content-Type: multipart/form-data" \
  -F "file=@your_dataset.csv" \
  -F "target_column=target"
